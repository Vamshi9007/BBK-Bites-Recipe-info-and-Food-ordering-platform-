# 🍽️ Food Ordering & Recipe Platform

## 📌 Description
A web-based Food Ordering and Recipe Platform built using HTML, CSS, and JavaScript. It includes login, home, and recipe modules, offering an interactive UI where users can explore recipes and understand ordering workflows.

---

## 🚀 Features
- 🔐 User Login Page
- 🏠 Home Page Interface
- 📖 Recipe Browsing
- 🛒 Food Ordering Simulation
- 📱 Responsive Design
- 🎨 Clean and User-Friendly UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

recipe-project/
│── login/
│── home/
│── recipes/
│── css/
│── js/
│── registration.html


---

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Navigate through login, home, and recipe pages
